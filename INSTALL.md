# Goal Prompt Coach Install

This folder is a repo-style Codex plugin marketplace bundle.

## Install

1. Unzip or clone this folder somewhere on your machine.
2. Add this folder as a local marketplace:

```bash
codex plugin marketplace add <path-to-this-folder>
```

3. Install the plugin:

```bash
codex plugin add goal-prompt-coach@dontcallmejames
```

4. Start a new Codex thread and try:

```text
Use goal-prompt-coach: make my app faster
```

## Included

- `.agents/plugins/marketplace.json`
- `plugins/goal-prompt-coach`
